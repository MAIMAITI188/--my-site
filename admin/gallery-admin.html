图库后台

现在可以直接选图、拖图、粘贴图片。图片会跟着 gallery.json 一起导出，不用先把图片手动丢到 images/gallery/ 里。

操作步骤

  1. 点击选图、拖图进来，或者直接 Ctrl+V 粘贴图片。
  2. 标题可以不填，系统会用文件名自动生成。
  3. 点“新增到列表”，最后下载 gallery.json。
  4. 把下载的文件覆盖到 data/gallery.json 就能生效。
推荐流程：默认用“快速贴图”。 只有在你已经自己把图片放进 images/gallery/ 时，才需要填“图片文件名”。
如果是双击本地 html 打开，“读取当前 gallery.json”可能会失败。但上传、导入、导出功能仍然可以用。
